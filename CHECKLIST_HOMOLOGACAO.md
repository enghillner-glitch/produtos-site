# Checklist de Homologação

## Portal do lojista

- [ ] Landing abre sem autenticação.
- [ ] Entrada simulada leva ao Dashboard.
- [ ] Dashboard mostra métricas agregadas.
- [ ] Estabelecimentos exibem elegibilidade.
- [ ] Wizard bloqueia estabelecimento não elegível.
- [ ] Wizard bloqueia envio sem validade inicial e final.
- [ ] Android Auto aparece desabilitado ou não publicável.
- [ ] Link HTTP é recusado.
- [ ] Link com encurtador é recusado.
- [ ] Enviar Alerta cria status `in_review`.
- [ ] Tela de Alerta criado mostra análise em andamento.
- [ ] Moderador aprova Alerta.
- [ ] Alerta aprovado pode ser ativado.

## Consumidor

- [ ] Vitrine mostra apenas Alertas ativos e aprovados.
- [ ] Vitrine respeita interesses selecionados.
- [ ] Alerta expirado não aparece.
- [ ] Alerta sem moderação aprovada não aparece.
- [ ] Oportunidade pode ser salva.
- [ ] Oportunidade pode ser ocultada.
- [ ] Oportunidade pode ser denunciada.

## Segurança e privacidade

- [ ] Lojista não vê histórico individual do consumidor.
- [ ] Métricas são agregadas.
- [ ] Link externo não abre diretamente por notificação.
- [ ] Android Auto não publica nada no MVP.
